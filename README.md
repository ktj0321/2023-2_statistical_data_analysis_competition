## 2023-2 통계 데이터 분석 공모전 출품작

2024년 가장 많은 사망 원인으로 꼽히는 질병군을 파악하고,  
각 질병군들로 인한 예측 사망자 수를 전통적인 시계열 분석법으로 예측해보았습니다.

분석 소프트웨어로는 R을 사용하였으며,  
**ARIMA 모형**과 **ARIMA 오차항을 가지는 회귀모형**을 이용한 예측이 주를 이룹니다.
